# GPStar

This repository consist of 2 API end points
  - Invoice creation (Post)
  - Invoice Update (Put)
  
 Invoice is consist of 
  - Date
  - Description
  - Total amount
  - Invoice Lines
  
 Invoice Line is consist of
  - Name
  - Quantity
  - Unit Price
  - Line amount
  
Running enviornment
 - AZ Cosmos DB emulator
 - Dot net 6.0
